### 方式1：在 Claude Code 中输入斜杠命令

1. 在 Claude Code 的输入框中输入：`/interview`
2. 按下回车或等待命令提示出现
3. 按照提示提供信息：

## 🔧 可能的问题和解决方案

### 问题1：斜杠命令不显示

**原因**：Claude Code 可能需要重新加载项目配置

**解决方法**：
1. 关闭并重新打开 Claude Code
2. 或者在输入框中输入 `/` 看看命令列表是否刷新
3. 如果还不显示，使用"方式2：直接对话"即可

### 问题2：权限被拒绝

**原因**：某些操作需要权限授权

**解决方法**：
- 首次使用时，Claude Code 会提示授权
- 请授权以下操作：
  - Bash（运行 Python 脚本）
  - Read（读取简历文件）
  - Write（生成输出文件）
  - Edit（编辑文件）
  - WebSearch（搜索公司信息）

### 问题3：简历找不到

**解决方法**：
- 确保简历路径正确
- 简历应该在 `InterviewIntel/resumes/` 目录下
- 或者直接提供完整的简历路径
